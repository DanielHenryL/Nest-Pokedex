## Description

[Nest] Pokedex con mondoDB y Docker. 

## Installation

```
$ npm install
```
## Ejecutar en desarrollo
1. Tener Nest CLI  
```
npm i -g @nestjs/cli
```
2. Levantar la base de datos
```
docker-compose up -d
```

## Stack 
* MongoDB
* Nest
* Docker


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
